학습 기록 Google Sheets 연동 방법

1

새 Google Sheets 생성

Google Drive에서 새 스프레드시트 생성 → 이름: 정보교과_학습기록

2

시트 ID 복사

URL에서 /d/[이 부분]/edit 복사

3

Apps Script 편집기 열기

스프레드시트 → 확장 프로그램 → Apps Script → learning_appscript.gs 내용을 붙여넣기

LEARNING_SPREADSHEET_ID 를 복사한 시트 ID로 교체

4

배포

배포 → 새 배포 → 웹 앱 → 액세스 권한: 모든 사용자 → 배포 → URL 복사

5

URL 입력

위 학습 기록 Google Sheets 연동 설정 란에 복사한 URL 입력 → 저장 & 로드

6

curriculum 파일 GS_URL 업데이트

배포 URL을 관리자에게 알려주면 모든 curriculum 파일의 GS_URL을 자동으로 업데이트해 드립니다.
